后台 UI 设计说明文档

---

## 1. 产品定位与设计目标

### 1.1 产品定位

* 类型：混合型 B 端业务后台（平台运营 + 商家）
* 行业属性：商品 / 库存 / 价格 / 订单 / 履约 / 数据
* 使用频率：中频使用，但单次操作深
* 核心页面类型：列表页（50%）

### 1.2 UI 设计目标

* **高信息效率**：减少点击、减少页面跳转
* **数据优先**：表格与筛选为核心
* **操作安全**：批量操作清晰、可确认
* **克制美学**：极简、干净、不干扰操作

---

## 2. 全局布局规范

### 2.1 整体布局

* 顶部：全局工具栏（高度 ≤ 56px）
* 左侧：主导航（固定，可折叠）
* 主内容区：页面内容

```
[ TopBar ]
[ Sidebar | Main Content ]
```

---

### 2.2 顶部工具栏（TopBar）规范

**仅包含全局行为，不包含任何业务操作**

包含元素：

* 系统 Logo / 系统名
* 语言切换（i18n）
* 主题切换（Light / Dark）
* 用户菜单（头像 + 下拉）

不允许出现：

* 业务按钮
* 页面级搜索

---

### 2.3 左侧主导航（Sidebar）

#### 一级菜单结构（固定顺序）

```
商品管理
库存管理
价格管理
订单管理
履约管理
数据中心
系统设置
```

#### 设计要求

* 支持折叠 / 展开
* 当前模块高亮
* 商家角色仅展示授权模块

---

## 3. 页面类型设计规范

### 3.1 列表页（核心页面类型）

#### 页面结构（固定模板）

```
[ 页面标题 ]
[ 页面说明（可选） ]

[ 筛选区（可折叠） ]

[ 操作区 ]

[ 数据表格 ]

[ 分页器 ]
```

---

#### 3.1.1 筛选区（Filter Area）

* 默认展示 3~4 个高频筛选条件
* 其余条件放入「更多筛选」
* 支持条件组合查询

常见筛选类型：

* 关键词搜索（商品名 / SKU / 订单号）
* 状态（多选）
* 时间区间
* 仓库 / 渠道 / 商家

操作要求：

* 查询按钮
* 重置按钮
* 支持回车触发查询

---

#### 3.1.2 操作区（Action Bar）

##### 默认状态（未选中行）

* 新增
* 批量导入
* 批量导出

##### 勾选状态（选中 ≥1 行）

* 显示「已选择 X 项」
* 批量操作按钮组

常见批量操作：

* 批量上架 / 下架
* 批量修改库存
* 批量修改价格
* 批量导出

**危险操作必须二次确认**

---

#### 3.1.3 表格（Table）规范

* 中等密度表格
* 第一列固定为 Checkbox
* 最后一列为操作列
* 表头支持排序（如适用）

示例字段：

* 商品名
* SKU
* 库存
* 价格
* 状态
* 更新时间

---

### 3.2 详情页（Detail Page）

#### 页面结构

```
[ 返回列表 ]    [ 主操作按钮 ]

[ 核心信息卡片 ]

[ Tabs ]
  - 基本信息
  - 库存信息
  - 价格信息
  - 操作记录
```

设计要求：

* 不使用长滚动
* 信息按 Tab 分类
* 主操作按钮固定位置

---

### 3.3 表单页 / 配置页

#### 表单规范

* 单列布局为主
* 字段分组（Section）
* 必填字段清晰标识
* 错误提示紧贴字段

按钮规范：

* 主按钮：保存
* 次按钮：取消 / 返回

---

### 3.4 数据看板（Dashboard）

#### 定位

* 仅做业务概览
* 不做复杂 BI

示例指标：

* 今日订单数
* 今日销售额
* 库存预警数
* 履约异常数

设计原则：

* 每个指标可点击
* 点击后跳转至对应列表页并携带筛选条件

---

## 4. 状态与反馈设计

### 4.1 状态展示

* 状态 = 文本 + 轻量色彩
* 禁止使用强对比大色块

示例：

* 正常 / 启用
* 停用
* 异常

---

### 4.2 操作反馈

* 成功：轻量 Toast
* 失败：明确错误原因
* 批量操作：展示处理结果摘要

---

## 5. 主题与视觉规范

### 5.1 视觉风格

* 极简、冷静、偏中性色
* 浅色为主，支持深色模式
* 边框弱化，依靠间距区分层级

### 5.2 字体与层级

* 页面标题 > 模块标题 > 正文
* 不超过 3 级视觉层级

---

## 6. 国际化与可扩展性

* 所有文本必须支持 i18n
* 不允许硬编码文案
* 支持多租户上下文切换
